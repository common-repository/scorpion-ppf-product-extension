== Description ==

This plugin allows you to add Scorpion Paint Protection Film products to your WordPress website. Your customers will be able to select their vehicle year, make, model, and trim. They will then be prompted with a selection of film products to purchase and add to their purchase. The product is then added to the cart and checkout can be made using normal processes.

Features
* Simple integration into a prepared WordPress / WooCommerce storefront
* Interaction with external 3rd party API system for fresh product variants

== Installation ==
1. Install and activate the plugin.
2. Configure the plugin by visiting Settings / Scorpion PPF area of your WordPress installation.
3. Follow the product setup instructions on the configuration page to complete your product setup.
 
Prerequisites (before usage)
* WooCommerce
* Gravity Forms
* Gravity Forms Product Add-On
